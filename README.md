# Bus_Reservation_System_JS
This project is a full-stack Bus Reservation System built using Node.js, Express, HTML, CSS, and JavaScript.

The system allows users to:
1. Book a seat
2. Cancel a booking
3. View passenger details
4. Check available seats
5. View seat layout

How to run the program:

1. Clone this repository in your workspace.
2. Install node.js and express.js.
3. In your terminal, type "node server.js".
4. Click on the link it provides.
5. Done.
